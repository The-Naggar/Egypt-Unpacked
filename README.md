Egypt Unpacked is a modern, interactive front-end platform designed to showcase Egypt's famous tourist attractions and events. It offers a centralized, visually appealing, and responsive solution for tourists looking to explore destinations, find events, and simulate ticket reservations.

This project was developed by the team: Omar Yasser Soliman, Ziad Mohamed AbdelRazik, Mazen Mohamed Fahr El Din, Ahmed Mohamed Mahmoud, and Shahd Magdy Abd El wahid.

‚ú® Features

Main Functionality (For Tourists)

Interactive Attractions: Explore destinations through engaging cards and descriptions.

Events Page: Browse upcoming events with search and filter functionality.

Simulated Ticket System: Reserve tickets for attractions and events (front-end simulation only).

Gallery: View images of key sites and access the ticket booking modal directly.

Contact Page: Submit inquiries via a form with basic front-end validation.

Key Highlights

Stunning Landing Page: Features a compelling 3D image of the Tutankhamun mask for a strong visual impact.

Responsive Design: Optimized for a smooth and intuitive user experience across mobile, tablet, and desktop devices.

Role-Based Authentication: (Conceptual/Future Feature) Mentioned in the functional requirements for potential expansion.

üöÄ Getting Started

To get a copy of the project up and running on your local machine for development and testing, follow these steps.

Prerequisites

You need to have Node.js and npm (or yarn) installed on your machine.

Node.js (which includes npm)

Installation

Clone the repository:

git clone [Your Repository URL Here]
cd egypt-unpacked



Install dependencies:

npm install
# or
yarn install



Run the application:

npm start
# or
yarn start



The application should now be running on http://localhost:3000 (or another port if 3000 is occupied).
